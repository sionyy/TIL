# CIL

## **CLI 정의 : Command Line Interface**

+ 터미널을 통해 사용자와 컴퓨터가 상호 작용하는 방식
- GUI는 그래픽을 통해 상호작용

---

## 터미널 명령어

1. touch
   
   + 파일을 생성하는 명령어
   - [예시] touch text.txt

2. mkdir
   
   - make directory
   
   - 새폴더를 생성하는 명령어
   
   - [예시] mkdir folder / mkdir 'sion folder'

3. ls
   
   + list segments
   
   + 현재 작업중인 디렉토리 폴더/파일 목록을 보여주는 명령어      
   
   + -a : all옵션 숨김파일까지 / -l : long옵션 자세히

4. mv
   
   + move
   
   + 폴더/파일 <U>이동 또는 변경</U>
   
   + [예시] mv text.txt folder
   
   + mv text1.txt text2.txt

5. cd
   
   + change directory
   
   + 디렉토리 변경
   
   + cd ~ : 홈디렉토리로 이동
   
   + cd .. : 상위 디렉토리
   
   + cd - : 바로 전 디렉토리
   
   + 절대경로 변경 가능 [cd C:/Users/sion]

6. rm
   
   + remove
   
   + 바로 완전 삭제

7. start
